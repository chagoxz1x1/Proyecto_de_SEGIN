Aplicaciones de cifrado.

  • • BitLocker

    BitLocker es una aplicación de cifrado de disco. esta diseñada para proteger los datos e impedir el acceso no autorizado a cualquier archivo almacenado en la unidad cifrada mediante una clave de 128 bits

    descargar

  • Cryptii •

    Cryptii es una aplicación web OpenSource bajo licencia MIT donde puedes convertir, codificar y decodificar contenido entre sistemas de diferentes formatos. Esto sucede completamente en su navegador usando JavaScript, ningún contenido será enviado a ningún tipo de servidor. Además, puede descargar y usar esta aplicación web sin conexión como se describe a continuación. Cualquier comentario apreciado, solo déjame un tweet o contribuye a este proyecto.

    descargar

  • • Cryptomator

    Cryptomator es una herramienta que cifra tus archivos y te permite sincronizarlos entre dispositivos a través de la nube o unidades USB de manera segura.

    descargar

  • • DiskCryptor

    DiskCryptor es una herramienta gratuita que nos va a permitir cifrar particiones de disco en nuestro sistema Windows.

    descargar

  • Enigmail •

    Enigmail es una extensión adicional para Mozilla y Mozilla Thunderbird para las versiones que funcionan bajo los sistemas Microsoft Windows y de tipo Unix, como GNU/Linux.

    descargar

  • • WinRar

    WinRAR es una versión del compresor RAR para Windows, una herramienta potente que le permite crear, manejar y controlar los ficheros de archivo. En la actualidad, existen varias versiones de RAR para varios sistemas operativos: Windows, Linux, FreeBSD, y macOS .

    descargar

  • • Whisply

    Whisply es una práctica aplicación web gratuita para el cifrado de archivos, antes de almacenarlos o compartirlos en Internet. Una herramienta imprescindible para los que se preocupan por su seguridad.
